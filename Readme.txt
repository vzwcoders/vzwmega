VZ Mega Usage Processing and Alerting System.
This system is designed for scalability and cloud ready.
System aims to enables to send the alerts in near realtime.

Key Components :
Core System - Process,Filter, Aggregate and Updat DB
Alerting System - Check the limits and send the alerts
Alerting System - View the alerts generated
Reporting - Reports to view the Usages processing and Alerting systems
Monitoring System - to Monitor the processes

We have used all the open source technologies
1. Java
2. Tomcat
3. MYSQL
4. D3.js
